# AGN-spectral-variability-analysis
# Анализ спектральной переменности AGN по данным SDSS и DESI

Проект предназначен для автоматизированной обработки многоэпоховых спектров активных галактических ядер по данным обзоров SDSS и DESI.

Основная цель проекта — скачать спектры объектов, вычесть вклад звёздного населения родительской галактики, измерить потоки широких компонент линий Hα и Hβ, оценить поток континуума на 5100 Å, выполнить нормировку по линии [O III] и определить наличие спектральной переменности.

---

## Общая схема работы

Пайплайн состоит из трёх основных этапов:

1. Скачивание спектров SDSS и DESI и вычитание звёздного населения родительской галактики.
2. Моделирование линий Hα и Hβ, измерение потоков широких компонент и континуума 5100 Å.
3. Нормировка по линии [O III], построение кривых блеска и определение переменности.
