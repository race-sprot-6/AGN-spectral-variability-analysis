# AGN-spectral-variability-analysis
# Анализ спектральной переменности AGN по данным SDSS и DESI

Проект предназначен для обработки многоэпоховых спектров объектов AGN/IMBH-кандидатов по данным обзоров SDSS и DESI.  
Код позволяет скачать спектры, вычесть вклад звёздного населения родительской галактики, измерить потоки широких линий Hα и Hβ, оценить поток континуума на 5100 Å, выполнить нормировку по линии [O III] и проверить наличие спектральной переменности.

## Основные скрипты

### `downloader-desi-sdss+pPXF.py`

Скрипт выполняет два основных этапа:

1. Скачивает спектры SDSS и DESI для объектов из входного файла.
2. Вычитает вклад звёздного населения родительской галактики с помощью пакета `pPXF`.

Входной файл должен содержать названия объектов в формате:

```text
J132523.35+593643.4
J160700.60+553809.2
J121540.80+615323.4

### Flux-Ha-Hb-Cont5100.py

Скрипт выполняет спектральное моделирование и измерение потоков:

1. моделирует области линий Hα и Hβ;
2. выделяет широкие компоненты линий;
3. подсчитывает потоки широких компонент Hα и Hβ;
4. сглаживает континуум;
5. оценивает поток континуума в области 5100 Å.

### normOIII+calc-variable.py

Скрипт выполняет финальную обработку результатов:

1. нормирует измеренные потоки по узкой линии [O III];
2. строит кривые блеска;
3. рассчитывает статистики переменности;
4. определяет, является ли объект переменным.
